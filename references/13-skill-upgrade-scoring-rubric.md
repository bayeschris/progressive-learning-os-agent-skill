# Skill Upgrade Scoring Rubric (Daily)

Score each shipped upgrade (0â€“5 per dimension).

## Dimensions
1. **Time Saved**
- 0: none
- 3: noticeable reduction in repeated effort
- 5: major recurring time savings

2. **Quality Gain**
- 0: no output improvement
- 3: clearer/fewer errors
- 5: materially better decisions or deliverables

3. **Risk Reduction**
- 0: no risk impact
- 3: mitigates common failure mode
- 5: prevents high-severity failure/path drift

4. **Reusability**
- 0: one-off only
- 3: reusable in similar tasks
- 5: broadly reusable across tracks/projects

## Composite score
- **Total = Time + Quality + Risk + Reusability** (max 20)

## Daily standard
- Ship 3 upgrades/day
- Target average score >= 12
- At least one upgrade/day must score >= 15
